ALPHA_VANTAGE_TIME_SERIES_INTRADAY

Description: Retrieves intraday OHLCV for a given equity.
Input Format:
adjusted (optional): Return adjusted values ('true' or 'false'). Default is 'true'.
datatype (optional): Response format ('json' or 'csv'). Default is 'json'.
extended_hours (optional): Include pre/post-market data ('true' or 'false'). Default is 'true'.
interval: Time interval between data points (e.g., '1min').
month (optional): Specific month in 'YYYY-MM' format.
outputsize (optional): Return latest 100 points ('compact') or full-length data ('full'). Default is 'compact'.
symbol: Equity ticker symbol (e.g., 'IBM').
Output Format: JSON object containing intraday time series data.

time_to (optional): End of time range (inclusive) in YYYYMMDDTHHMM format. Default is current time.

Output Format: JSON object containing news sentiment data.
